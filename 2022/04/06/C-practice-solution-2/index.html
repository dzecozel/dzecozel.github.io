C practice solution (2)

C语言每日隔几天刷题（2）

【问题描述】

现有26名同学(使用‘A’~‘Z’进行标记)围坐一圈。从键盘输入一个正整数k(1<=k<=10)，现在从标记为‘A’的同学开始报数如果所报数字能够被k整除，则该同学退出圈中，剩余同学重新开始游戏。报数的规则如下：

1)每一次开始游戏均从1开始报数;

2)每一名同学所报数字为前一个同学所报数字加1。

请按照同学出圈的顺序输出其标记。

【输入形式】

1个正整数。

【输出形式】

26个字母构成的字符串。

【样例输入】

8

【样例输出】

HPXFOYISCNAMBRGWTLKQVEDUJZ

【思路分析】

  1. 先建立两个26位的字符串，用于储存二十六个字母，一个用于判断，一个在处理时使用。

  2. 对用于处理得组别进行数字同步1~k编号，然后对数字进行处理，若该字符对应的数字为k，则输出字符并将用于判断中的该字符化为NULL。

    注意：a. 对应字符最后一位不一定是k，需要单独留一个数来保存字符。

    ​ b. 为了能够作用于判断用字符串，也需要单独记录下对应数字。

  3. 判断函数ded

  1. 总结：具体思路

  2. 如果字符串中仍有非空字符则排序，去除已经改为空的字符。

  3. 对已有字符进行1~k编号。

  4. 输出k代表的字符，并将该字符转化为NULL.

总结：难度一般，但是思路需要清晰，尤其是处理字符和标号一一对应的关系。另外需要注意添加标记字符。（能不能用指针实现？）
